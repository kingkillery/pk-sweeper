## Summary

Keep open. Current `main` has not absorbed this PR's Windows test updates, and the PR is not a safe auto-close candidate because it also changes runtime behavior in `expandHomePrefix`, `isExecutableFile`, and `resolveStableNodePath`. The supplied review discussion already identified plausible regressions in those runtime edits, so conservative cleanup should not close it as implemented, unreproducible, or stale.
