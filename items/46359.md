## Summary

Keep open. Current `main` already replaced the PR's Ollama-side static hostname checks with stronger SSRF-guarded, DNS-pinned fetches, but the shared OpenAI-compatible local discovery path used by vLLM and sglang still performs a raw `fetch()` with the configured `baseUrl`. The request is only partially implemented on `main`, so there is not strong evidence to close this PR.
