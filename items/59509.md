## Summary

Keep open. The request is still a plausible feature gap on current main: google-vertex is still only routed as a Google provider hook alias, the Google plugin manifest does not register google-vertex as a provider, and the model/provider config types do not expose a model-level location field. A related implementation PR (#60860) exists and explicitly targets Vertex ADC/global endpoint routing, but it is still open rather than merged.
