## Summary

Keep open. Current `main` still resolves memory status by instantiating the active memory plugin in the local CLI process, so the core behavior described in #62599 has not been removed. Main includes partial `status`-purpose mitigations inside bundled `memory-core`, but the status command still does not ask the live gateway for memory/vector truth and still loads plugin runtime locally.
