## Summary

Atomic Chat is a useful optional model-provider integration, but current main already has the plugin and provider SDK surfaces needed to ship it externally. The repo vision and docs direct optional plugins to ClawHub/npm unless there is a missing core API or maintainer-owned reason to bundle it.
