## Summary

Keep open. Current `main` does not implement this hardening: the managed session store, transcript, setup/onboarding, agent-creation, and probe paths still create session directories with plain recursive `mkdir`, while the PR discussion shows substantive security/behavior gaps being iterated rather than a fix already landing elsewhere.
