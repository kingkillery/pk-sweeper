## Summary

Keep open. Current `main` already covers the follow-up routing bug generically and already has native-button guidance in the agent system prompt, but the approval-pending tool message is still CLI-only and has no `turnSourceChannel` plumbing. That means this PR is only partially implemented on `main`, so there is not strong evidence to close it as redundant.

## Evidence
