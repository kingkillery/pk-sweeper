## Summary

Keep open. Current `main` does not show Termux as a supported OpenClaw host/runtime. The docs explicitly position Android as a companion node while requiring the Gateway on macOS, Linux, or Windows via WSL2, and the only Termux references in-tree are optional helper scripts for auth monitoring workflows rather than core Termux installation/support.
