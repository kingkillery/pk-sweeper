## Summary

Keep open. Current `main` still uses a fixed 100-message `chat.history` window for subagent announce extraction, still returns mixed-history raw tool text before any assistant-only fallback, and the fallback helper itself reads the same limited window. Docs and tests still encode tool/toolResult fallback behavior, and there is no evidence of a fix on `main` or since the latest release.
