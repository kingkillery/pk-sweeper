## Summary

Keep open. Current `main` does not implement `POST /hooks/message`; hook handling still only covers `wake`, `agent`, and mapped custom subpaths, and `/hooks/agent` still has main-session fallback side effects that match the issue's boundary concern. The issue also already links an open implementation PR (#62528), so this is not a conservative cleanup close.
