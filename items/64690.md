## Summary

Keep open. Current `main` still requires a channel `sendPayload` implementation to deliver interactive approval payloads, but the Microsoft Teams plugin does not expose or forward `sendPayload`; it only wires `sendText`, `sendMedia`, and `sendPoll`. The report is recent, specific, and already linked to an open PR, so there is no conservative auto-close basis.
