## Summary

Keep open. Current `main` does not contain either behavior this PR proposes, so there is no basis to close it as already implemented or obsolete. The UI still initializes locale loading asynchronously after construction and renders immediately, and the agent error sanitizers still lack a dedicated `tool_use`/`tool_result` mismatch recovery path.
