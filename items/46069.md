## Summary

Current `main` does not include this PR's `encoding_format: "float"` change in `extensions/memory-lancedb`, and the bundled `openai` dependency used by that plugin still defaults embeddings requests to `base64` when `encoding_format` is omitted. The existing test suite in `main` also still asserts the old request shape without `encoding_format`, so there is not strong evidence that this PR is obsolete or already implemented.
