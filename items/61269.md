## Summary

Keep PR #61269 open. Current main still has the Android onboarding finish gate require both operator connectivity and node connectivity, and the current unit test still asserts that node-only connectivity must block finish. That means the observable bug described in linked issue #61005 remains present on main, and this PR is not obsolete or already implemented.
