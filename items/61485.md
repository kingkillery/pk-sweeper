## Summary

Keep open. Current main still treats llm_input and llm_output as observe-only fire-and-forget hooks, while #61485 proposes a missing core plugin API extension that would let plugins block, rewrite, or redact LLM traffic. This is not a ClawHub-only cleanup because the capability cannot be implemented by a plugin until core exposes and consumes modifying hook results.
