## Summary

Keep open. Current `main` still does not implement `agents.defaults.reasoningDefault`; only per-agent `agents.list[].reasoningDefault` exists, and runtime resolution still falls back to `off` when no session or per-agent override is set.

## Evidence

- **Global defaults type omits reasoningDefault:** `AgentDefaultsConfig` includes `thinkingDefault`, `verboseDefault`, and `elevatedDefault`, but no `reasoningDefault` field, so the requested top-level config key is not part of the typed config contract.
  - file: [src/config/types.agent-defaults.ts:297](https://github.com/openclaw/openclaw/blob/ed6094b3014c56882da15a7942130b00ea410c6e/src/config/types.agent-defaults.ts#L297)
