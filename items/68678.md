## Summary

Keep open. Current `main` does not implement LM Studio native inference via `/api/v1/chat`; the bundled LM Studio provider still normalizes inference to `/v1` and documents `/v1/chat/completions` as the default inference path. The request is coherent and actionable, and the plugin SDK already has a provider-native transport seam for this kind of work, so there is no strong conservative-close reason.
