## Summary

Keep open. Current `main` at `c997a9f9788e0a96e8eab05e7ae94959e76cdb4a` still passes `abort?.signal ?? new AbortController().signal` into `plugin.gateway.stopAccount`, so the PR's one-line fix is not implemented on `main` and the inconsistency described in the PR remains real.
