## Summary

Keep open. Current `main` still implements model allowlisting only at the global `agents.defaults.models` level. Per-agent config supports `agents.list[].model` and `agents.list[].params`, but there is no per-agent `models`/allowlist field, and the allowed-model resolution path still reads only `agents.defaults.models`.
