## Summary

Keep open. Current `main` already moved Linux gateway installs onto systemd `EnvironmentFile=` handling and added parsing/tests around that path, but it still does not remove the generated `gateway.systemd.env` file on uninstall. That means the PR's cleanup behavior is not fully implemented on `main`, so there is not strong enough evidence to close it as already done.
