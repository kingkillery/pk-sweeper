## Summary

Keep open. Current `main` does not implement per-skill `thinking` or `model` overrides for `skills.entries.<skill>`. The config schema, gateway update API, docs, UI, and runtime wiring only support `enabled`, `apiKey`, `env`, plus generic `config`, and the runtime only consumes the first three for skill behavior.

## Evidence
