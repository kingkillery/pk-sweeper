## Summary

PR #67033 is superseded by current `main`: the Discord gateway plugin on `main` already includes the guarded metadata fetch path and the heartbeat timeout cleanup this PR was trying to unblock, and the Carbon 0.16.0 dependency contract now exposes `firstHeartbeatTimeout`, so the local structural typing workaround is no longer needed.
