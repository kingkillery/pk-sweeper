## Summary

Current main partially supersedes this PR, but not strongly enough to auto-close it. The old copy-paste permissions block is gone from docs/channels/feishu.md because the page was rewritten around the QR-based setup wizard, yet the reachable manual fallback note in extensions/feishu/src/setup-surface.ts still tells users to enable the older shorthand scopes. That means the PR's core docs concern is not clearly implemented on main, and closing it as implemented would be too aggressive.
