## Summary

Keep open. Current `main` still returns `listMemoryCorpusSupplements()` in raw insertion order, while registration and loader snapshot/cache paths still preserve that order. The deterministic sort and regression tests described in PR #63146 are not present on `main`, so this cleanup has not already been implemented.
