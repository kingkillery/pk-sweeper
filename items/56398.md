## Summary

Keep open. Current `main` still expires pending approvals with `"no-approval-route"` before the two-phase `accepted` response when there are no approval clients, no turn-source route, and no forwarded delivery. The behavior described in PR #56398 is not implemented on `main` at `7d1891e6e63dc35dd9dcb688ba308049e68fe695`.
