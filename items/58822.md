## Summary

Current main still appears to match the reported regression: both the normal subagent resolver and isolated cron resolver check the parent/agent model before `agents.defaults.subagents.model`. The issue is recent, concrete, corroborated by a user comment, and linked to an open PR (#58823), so it should stay open for maintainer review or fix landing.
