## Summary

Keep open. Current `main` does not implement the requested Control UI file-explorer toggle or a general workspace tree. The only related shipped surface is an Agents "Core Files" editor backed by an allowlisted bootstrap-file API, so this issue is still a coherent feature request rather than something conservatively auto-closeable.
