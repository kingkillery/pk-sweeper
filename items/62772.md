## Summary

Keep open. Current `main` has a narrow experimental local-model prompt trim and separate manual knobs for prompt override, tool policy, compaction, timeout, and local embeddings, but it does not implement the requested top-level `localMode` switch or an equivalent one-switch macro across those settings. The issue discussion is concrete and recent, with reporter-supplied before/after measurements and working manual mitigations, so this is not stale or incoherent.

## Evidence

- **Recent partial mitigation, not the requested top-level switch:** `CHANGELOG.md` records a new experimental `agents.defaults.experimental.localModelLean: true` flag for weaker local-model setups. The docs also say this is intentionally experimental and not a stable default-mode setting.
  - file: [CHANGELOG.md:608](https://github.com/openclaw/openclaw/blob/c997a9f9788e0a96e8eab05e7ae94959e76cdb4a/CHANGELOG.md#L608)
