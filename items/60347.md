## Summary

Keep PR #60347 open. Current main at 8f49c59d6daf3332df19e7c5ec8f3f354d307e42 does not implement the requested `memorySearch.local.gpu` config or pass a GPU backend through to `node-llama-cpp`. The PR discussion also contains useful production verification on Orange Pi 5 Plus/PanVK, but review comments identified integration gaps that are not solved by current main. This remains a meaningful open contribution rather than conservative cleanup material.
