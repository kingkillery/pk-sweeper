## Summary

Current `main` does not implement the PR’s rescue-watchdog feature set. Some adjacent daemon/service pieces exist, but `main` still lacks a `rescueWatchdog` cron payload, rescue runner wiring, and the abortable restart/probe behavior this PR adds, so there is no strong basis to close it as already implemented or obsolete.
