## Summary

Keep open. Current `main` already exposes Pi's built-in `ls` tool for directory listing, but OpenClaw still does not ship a first-class `list` tool or include directory enumeration in `group:fs`/tool-catalog docs, so this PR is not implemented on `main`. The proposed PR also has unresolved correctness/security review blockers, so it should not be conservatively closed as duplicate or fixed.
