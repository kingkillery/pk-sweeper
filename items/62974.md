## Summary

Keep open: current `main` still does request-scoped model fallback with cooldown probes, but it does not implement any cross-request sticky fallback or cache-aware provider pinning window. The reported behavior remains plausible and the requested control surface is not present in code or config.

## Evidence

- **Fallback state is per run, not persisted across future requests:** `runWithModelFallback(...)` creates local `attempts` and `cooldownProbeUsedProviders` state, iterates fallback candidates for that invocation, and returns on success. There is no persisted provider-stickiness window here.
  - file: [src/agents/model-fallback.ts:715](https://github.com/openclaw/openclaw/blob/245451b6a9c529464e177afa65013adca0d63eb5/src/agents/model-fallback.ts#L715)
  - command: `nl -ba src/agents/model-fallback.ts | sed -n '660,860p'`
  - sha: [245451b6a9c5](https://github.com/openclaw/openclaw/commit/245451b6a9c529464e177afa65013adca0d63eb5)
- **Docs describe per-candidate cooldown probing:** The failover docs say cooldown handling is a per-candidate decision and that transient cooldown probes are limited to one per provider per fallback run, which matches request-scoped behavior rather than sticky cross-request failover.
  - file: [docs/concepts/model-failover.md:274](https://github.com/openclaw/openclaw/blob/245451b6a9c529464e177afa65013adca0d63eb5/docs/concepts/model-failover.md#L274)
  - command: `nl -ba docs/concepts/model-failover.md | sed -n '274,290p'`
  - sha: [245451b6a9c5](https://github.com/openclaw/openclaw/commit/245451b6a9c529464e177afa65013adca0d63eb5)
- **Auth cooldown config has no sticky-fallback knob:** `AuthConfig.cooldowns` exposes billing/auth/overloaded/rate-limit controls only. There is no `stickyFallbackMs`, cache-TTL-derived mode, or similar cross-request fallback-pinning option in the typed config.
  - file: [src/config/types.auth.ts:14](https://github.com/openclaw/openclaw/blob/245451b6a9c529464e177afa65013adca0d63eb5/src/config/types.auth.ts#L14)

  - sha: [245451b6a9c5](https://github.com/openclaw/openclaw/commit/245451b6a9c529464e177afa65013adca0d63eb5)
