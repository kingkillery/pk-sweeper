## Summary

Keep open. Current `main` does not implement the PR's proposed `sessions_spawn` API (`contextFiles` / `inheritContext`), and the existing spawn workspace behavior intentionally uses the target agent workspace for cross-agent spawns rather than the requester workspace described in the PR.
