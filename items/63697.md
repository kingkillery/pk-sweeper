## Summary

Keep open. Current `main` still intercepts `/new`, `/reset`, `/clear`, and `/compact` locally in the shared chat view model and routes them through session reset/compact helpers rather than `chat.send`, so the PR's core behavior is not implemented on `main` and is not obsolete.
