## Summary

Keep open. Current `main` still routes Discord outbound text through the same chunking path, and that path only preserves fenced code blocks and a special reasoning-italics case. I did not find current-main handling or regression tests for blockquotes, nested lists, tables, or inline code spans, so there is not strong evidence that #63409 is already fixed.
