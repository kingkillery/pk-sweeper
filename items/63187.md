## Summary

Keep open. Current `main` still tracks token/cache/cost usage, but it does not expose a dedicated TTFT metric in embedded run metadata or documented usage surfaces. There is a partial workaround for gateway consumers via top-level agent-event timestamps, but the pi embedded subscription callback used for streamed agent events still drops that timing data and even strips `startedAt` from the lifecycle start callback, so the issue is not implemented as requested.
