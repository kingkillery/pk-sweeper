## Summary

Keep open. Current `main` (c0a7b6a510758645c5267734b984f4885a605b67) and the latest release `v2026.4.22` still implement acknowledgment as emoji reactions and typing/status indicators, not a configurable pre-processing text acknowledgment. I found no `ackText`-style config or send path on `main`, so the feature request is not implemented and is still actionable.
