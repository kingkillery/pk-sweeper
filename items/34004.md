## Summary

Current main does not implement the requested Discord self-profile action. It has a narrower `set-presence` action for status/activity only, behind `channels.discord.actions.presence`, but no `self-profile`/`selfProfile` gate, nickname update, avatar update, self-only target guard, or best-effort profile mutation path. The related PR #34010 was closed unmerged, so it does not supersede the open issue.
