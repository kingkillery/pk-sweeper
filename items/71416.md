## Summary

Keep open. Current `main` still unconditionally catches gateway failures in `openclaw agent`, prints only a stderr diagnostic, and runs the embedded path without any opt-in flag, transport discriminator, or non-zero exit handling.

There is no guard, no exit-code change, and no transport marker added here.
