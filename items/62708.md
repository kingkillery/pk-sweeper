## Summary

Keep open. Current `main` still defines, writes, preserves, and reads `claudeCliSessionId`, and recent tests on `main` explicitly cover legacy fallback behavior. The issue’s core claim that no active read path depends on the field is no longer true on `main`, so there is not strong evidence for a conservative close.
