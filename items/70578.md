## Summary

Keep open. Current main does not match the report's first two mechanisms as written: unknown root keys still fail validation instead of being silently kept from partial Zod output, and the config write projection now preserves source-only keys absent from the runtime shape. But bundled CLI commands are still intentionally gated by `plugins.allow`, and I did not run a live `plugins install` reproduction in this checkout, so conservative cleanup should not auto-close this issue.
