## Summary

Keep PR #61335 open. Current main still contains the DST-unsafe Usage hour boundary logic the PR targets, and a read-only Node reproduction shows the loop can stall at the 2025 DST fallback boundary in America/New_York. The PR discussion also contains a Greptile review explicitly supporting the proposed DST-safe monotonic stepping approach.
