## Summary

Keep open. Current `main` does not implement the PR's `rolling` context-pruning surface, so this is not `implemented_on_main`. At the same time, the metabolism half is not a clean `clawhub`/scope-only close: the repo's current plugin contracts require a real plugin entrypoint, and the PR discussion already moved toward splitting/reducing scope after review feedback. This needs maintainer triage or a split PR, not conservative auto-closure.
