## Summary

Do not close. Current `main` still matches the report: `src/agents/tools/pdf-native-providers.ts` uses raw `fetch()` for both PDF providers, Anthropic still sends `x-api-key` to a user-controlled `baseUrl`, and Gemini still places the API key in the `?key=` query string. The PR is also not ready to close as 'implemented on main', and its current strict SSRF-guard approach would regress proxy-only deployments because strict guarded fetch intentionally ignores env proxy routing.
