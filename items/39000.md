## Summary

Current `main` still lacks any plugin-facing gateway invocation API. The public `OpenClawPluginApi` exposes `registerGatewayMethod(...)` but no `callGatewayMethod(...)`, the SDK docs list registration only, and the trusted plugin runtime surface likewise has no generic gateway-call helper. There is internal gateway dispatch code in `src/gateway/server-plugins.ts`, but it is currently used for plugin subagent runtime plumbing rather than exposed as a supported inter-plugin contract.

## Evidence
