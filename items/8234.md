## Summary

Keep open. Current main supports local models through external runtimes such as Ollama, LM Studio, and OpenAI-compatible local proxies, plus optional local memory embeddings via `node-llama-cpp`, but there is no Intel AMX-specific implementation, configuration surface, or documentation on main. This is still a plausible enhancement request, not a strong fit for any close bucket.
