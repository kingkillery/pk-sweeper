## Summary

Current `main` still has the reported mismatch: `sessions_spawn` publishes a schema that always allows `streamTo`, but the implementation rejects `streamTo` unless `runtime="acp"`. The inconsistency is also reflected in docs/tests, so this is not closeable as already fixed or obsolete.
