## Summary

Current `main` does not implement a per-run tool allowlist for `sessions_spawn` / spawned subagents. The spawn tool schema and `SpawnSubagentParams` expose model/thinking/timeout/thread/context controls, but no `toolsAllow`-style field, while the only documented subagent tool restriction is the static config-level policy at `tools.subagents.tools.allow` / `deny`. A separate per-run allowlist already exists for embedded/cron agent turns, so this issue is still a real feature gap rather than already implemented behavior.
