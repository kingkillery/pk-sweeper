## Summary

Keep #60982 open. Current main does not show an equivalent Discord final-delivery sanitizer: Discord final replies are forwarded into the shared outbound delivery path without a Discord sanitizer, and the shared outbound path only sanitizes when the channel adapter provides `sanitizeText`, which Discord does not. The PR also has meaningful review discussion about edge cases, so it should remain available for maintainer review rather than being closed as already implemented.
