## Summary

Keep open. Current main still lacks the requested source-based autoCapture filter for the LanceDB memory plugin: config and manifest schemas do not accept captureFilter/excludeWrapped, and the agent_end auto-capture path still feeds user text directly into shouldCapture(), whose triggers include always/never/important without checking EXTERNAL_UNTRUSTED_CONTENT wrappers first.

## Evidence
