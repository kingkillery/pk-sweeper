## Summary

Current `main` only partially matches PR #46766. The chat context notice already uses `session.totalTokens`, but the `/usage` slash command still computes context percentage from `session.inputTokens`, and the current test suite locks in that incorrect behavior. This PR still carries a real, not-yet-implemented fix, so it should stay open.
