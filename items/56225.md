## Summary

Keep open. Current `main` already has shared `audioAsVoice` plumbing, but WeChat remains an external plugin and native TTS voice-note support in core still excludes WeChat. The report is still plausible and not implemented on `main`, so there is not strong evidence to close.

## Evidence

- **WeChat integration is external-plugin owned:** The repo docs state that `@tencent-weixin/openclaw-weixin` is an external channel plugin and that WeChat-specific runtime, login, media upload/download, and monitoring are owned by that external plugin.
  - file: [docs/channels/wechat.md:10](https://github.com/openclaw/openclaw/blob/7d1891e6e63dc35dd9dcb688ba308049e68fe695/docs/channels/wechat.md#L10)

  - file: [src/auto-reply/reply-payload.ts:28](https://github.com/openclaw/openclaw/blob/7d1891e6e63dc35dd9dcb688ba308049e68fe695/src/auto-reply/reply-payload.ts#L28)
