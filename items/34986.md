## Summary

Current main does not implement the requested configurable heartbeat pre-flight script. The related PR #35504 usefully captured the intended shape (`agents.defaults.heartbeat.preFlightCommand`) but was closed unmerged, and current code/docs only provide lighter heartbeat modes that still create an agent turn.
