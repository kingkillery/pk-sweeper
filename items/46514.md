## Summary

Keep open. Current `main` still treats Discord guild channels as agent-scoped sessions, and the Discord inbound path reads/writes session context from the routed agent’s own store. The only channel-level history injected in Discord is the ephemeral `guildHistories` buffer for pending/skipped messages, not shared cross-agent transcript replay, so the reported limitation is still real rather than already fixed.

## Evidence
