## Summary

Keep open. Current main contains the related run-to-attempt `toolsAllow` forwarding, but it does not implement this PR's config-policy fix: `toolsAllow` is still applied only after `createOpenClawCodingTools`, so a restrictive config allowlist can remove core tools before the final post-filter runs.
