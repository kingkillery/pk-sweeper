## Summary

Keep open. Current `main` docs now steer `before_tool_call`/`after_tool_call` to plugin hooks, but managed/internal `HOOK.md` loading still accepts arbitrary event strings, registers them only in the internal-hook registry, and `openclaw hooks info` still reports readiness from config/requirements rather than dispatch reachability. Tool-lifecycle dispatch continues to use the global plugin hook runner instead of `triggerInternalHook`, so the silent no-op described in the issue still has a real code-path basis.

## Evidence

- **Managed hooks still register any declared event into the internal-hook registry:** `loadInternalHooks()` iterates `entry.metadata?.events` and calls `registerInternalHook(event, handler)` for each string without validating whether the event is actually dispatchable by the internal hook system.
  - file: [src/hooks/loader.ts:155](https://github.com/openclaw/openclaw/blob/c7f18a6b9d76a31f4daed5096c0a25b3794c76ac/src/hooks/loader.ts#L155)
  - command: `nl -ba src/hooks/loader.ts | sed -n '130,190p'`
  - sha: [c7f18a6b9d76](https://github.com/openclaw/openclaw/commit/c7f18a6b9d76a31f4daed5096c0a25b3794c76ac)
- **Internal hook event types do not include tool lifecycle events:** The internal hook event union is limited to `command | session | agent | gateway | message`; there is no internal `before_tool_call` or `after_tool_call` event type.
  - file: [src/hooks/internal-hook-types.ts:1](https://github.com/openclaw/openclaw/blob/c7f18a6b9d76a31f4daed5096c0a25b3794c76ac/src/hooks/internal-hook-types.ts#L1)
  - command: `nl -ba src/hooks/internal-hook-types.ts | sed -n '1,80p'`

- **Internal dispatch only fans out by internal event type/action:** `triggerInternalHook()` only looks up handlers for `event.type` and `${event.type}:${event.action}` from `InternalHookEvent`, so handlers registered under bare keys like `before_tool_call` are unreachable from this dispatcher.
  - file: [src/hooks/internal-hooks.ts:286](https://github.com/openclaw/openclaw/blob/c7f18a6b9d76a31f4daed5096c0a25b3794c76ac/src/hooks/internal-hooks.ts#L286)

  - file: [src/agents/pi-tools.before-tool-call.ts:233](https://github.com/openclaw/openclaw/blob/c7f18a6b9d76a31f4daed5096c0a25b3794c76ac/src/agents/pi-tools.before-tool-call.ts#L233)
