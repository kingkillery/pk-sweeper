## Summary

Current `main` still unconditionally scans `~/.agents/skills`, and the exposed `skills.load` config only supports `extraDirs`, `watch`, and `watchDebounceMs`. There is no implemented opt-out such as `scanAgentsSkills` or `ignoreDirs`, so the requested coexistence toggle is not implemented on `main`.

## Evidence
