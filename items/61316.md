## Summary

Keep open. Current main still uses the custom truthy/falsy arrays directly as Sets, so custom entries remain case-sensitive and malformed non-iterable options can still throw. The PR’s requested behavior and regression tests are not implemented on main. Related PR #59535 is an older, narrower attempt and does not supersede this PR.
