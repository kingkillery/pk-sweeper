## Summary

Keep open. The linked PR #59344 appears intended to fix this, but the issue is still open and current main at 2e0ae56 does not show path-based deduplication after bootstrap hook overrides. The current sanitization only validates/coerces each file path and pushes every valid entry through, so duplicate paths introduced by hooks would still be preserved before context assembly.
