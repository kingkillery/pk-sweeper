## Summary

Keep open. Current `main` still returns the generic compaction-buffer advice for context overflow, and the heartbeat run path still persists the active model/context window back into session state without a heartbeat-specific guard. The linked PR #67381 is still open and unmerged, so the misleading error text is not implemented on `main`.
