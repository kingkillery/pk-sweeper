## Summary

Keep open. Current main has partial Instances i18n wiring, but the open PR still covers meaningful remaining work: scope/unknown fallback labels are still hardcoded in English, and the empty reason interpolation risk noted in review still exists on main.

## Evidence

- **Current main still hardcodes Instances fallbacks and scope labels:** `renderEntry` still uses `entry.mode ?? "unknown"`, `entry.host ?? "unknown host"`, and constructs scope chips as raw English strings (`"scopes"`, `"scopes:"`) instead of locale-backed strings.
  - file: [ui/src/ui/views/instances.ts:66](https://github.com/openclaw/openclaw/blob/b6af40f1f11e08d3c73d79c678351306bef4acbe/ui/src/ui/views/instances.ts#L66)
