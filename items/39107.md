## Summary

Keep open. Current `main` partially supports plugin-authored completions through the public plugin SDK, but it does not provide the requested direct CLI completion command or a root-package `openclaw.completion(...)` API, and there is no Python interface. The request also cuts across current scope guidance, but not cleanly enough to close as `clawhub`.
