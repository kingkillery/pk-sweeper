## Summary

Keep open. Current `main` still does not accept `models.providers.ollama.baseURL` in the shipped Ollama discovery path, and it still reads only `baseUrl` in embedding and web-search code paths. The PR remains useful, and the discussion already identifies an unresolved discovery-entry gap rather than an obsolete or already-landed fix.
