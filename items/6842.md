## Summary

Keep open. Current `main` only documents and ships same-gateway/session agent-to-agent routing, not the requested external A2A protocol with Agent Cards and JSON-RPC. The plugin system looks capable of hosting this as a plugin, but `VISION.md` is not explicit enough to close it as out of scope, and the issue already has an open related PR in the provided timeline.
