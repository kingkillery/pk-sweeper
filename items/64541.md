## Summary

Keep open. Current `main` still sends the full system prompt and tool schema surface to native Ollama requests without a shipped cold-start mitigation, the timeout surface reported by the issue is still present, the docs only document manual pressure-reduction workarounds, and the linked fix PR #66511 is still open/unmerged.
