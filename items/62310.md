## Summary

Keep open. Current `main` still lacks the proposed heartbeat gating/overdue-timer changes, so the underlying problem appears real, but this PR has an unresolved behavior risk: its blanket non-interval throttle would also delay explicit `wakeMode: "now"` flows that current code and docs treat as immediate.

## Evidence
