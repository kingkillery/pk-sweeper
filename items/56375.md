## Summary

Keep open. Current `main` still implements Control UI uploads as image-only attachments, so the broader "Web UI interface file upload" request is not implemented on `main`. The report is underspecified, but it is still a plausible feature request or UX bug rather than a strong close candidate.

## Evidence
