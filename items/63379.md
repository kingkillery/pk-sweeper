## Summary

Keep open. Current `main` still intentionally rejects loopback-source `trusted-proxy` requests, with tests and docs locking that behavior in, so the request is not implemented on `main`. The PR discussion also contains unresolved correctness/security concerns in the proposed fix, so this is not a safe conservative close.
