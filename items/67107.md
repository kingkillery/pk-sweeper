## Summary

Keep open. Current `main` still requires Node `>=22.14.0`, the installer still auto-installs Homebrew `node@24` on macOS, and the public install docs still describe macOS generically without a Big Sur/Monterey floor or a Big Sur-specific installer error path. The maintainer comment narrows this to installer/docs UX, but that UX gap is still present on `main`.
