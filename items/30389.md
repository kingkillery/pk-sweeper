## Summary

Keep open. Current main partially implements the requested WebChat UX: chat has an attachment button and a per-session model dropdown. However, the requested chat-level skill/capability menu is still not present, and file upload support is limited to images rather than the broader document/file support requested. This is a real remaining feature request, not a conservative cleanup close.
