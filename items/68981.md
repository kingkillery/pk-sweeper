## Summary

Keep open. Current `main` has partial progress on the subagent-cleanup part and already supports manual session labels, but the issue's requested `/session rename` or auto-title flow and richer Control UI grouping/filtering by channel/kind/status are still not implemented end-to-end.

That gives a manual naming workaround, but it is not a `/session rename` command and does not rename the session key itself.
