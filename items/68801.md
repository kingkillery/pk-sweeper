## Summary

Keep open. Current `main` does not contain the proposed `agentRunStarts` TTL sweep, and the PR as written still has an unresolved correctness risk: `agent.wait` falls back to `agentRunStarts` when terminal lifecycle events omit `startedAt`, so pruning active entries by age can drop `startedAt` for long-running runs.

## Evidence
