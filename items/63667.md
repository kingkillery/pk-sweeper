## Summary

Keep open. Current main and release `v2026.4.23` contain related media-understanding fixes around `agents.defaults.imageModel` and MiniMax VL fallback routing, but the exact reported path using explicit `minimax/MiniMax-M2.7-highspeed` is not clearly proven fixed. The generic image runtime still builds non-OpenRouter multimodal requests with the prompt in `systemPrompt` and image-only user content, which is still plausibly compatible with the reported `chat content is empty` failure on Anthropic-compatible MiniMax routes.
