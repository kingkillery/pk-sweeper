## Summary

Keep open. Current main at b6af40f1f11e08d3c73d79c678351306bef4acbe still has the pre-PR usage behavior: channel and agent rankings are cost-first, sessions.usage aggregates are built only from limitedEntries, no overview scope is returned, and the UI still recomputes insight duration from the returned session subset.
