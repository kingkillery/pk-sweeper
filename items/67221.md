## Summary

Keep open. Current `main` does not implement the email-based Google Chat add-on principal check described in PR #67221, so this is not `implemented_on_main`. At the same time, `main` still documents/tests `appPrincipal` as a numeric binding and warns when it is an email, so the PR is not strong enough for conservative cleanup as an obvious obsolete item. The PR’s secondary status-snapshot concern is also at least partly covered on `main` already via the Google Chat status adapter.

## Evidence
