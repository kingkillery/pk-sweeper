## Summary

Keep open. Current `main` only partially overlaps the request: the Control UI can preview/download explicit `MEDIA:` attachments and exposes a Files tab for agent bootstrap/core files, but there is no native arbitrary workspace tree/browser for `~/.openclaw/workspace/` media and files as requested.
