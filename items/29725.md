## Summary

Keep open. Current main has Discord presence plumbing for static config, auto-presence, and the Discord message action `set-presence`, but I did not find the requested supported code-plugin API `api.runtime.channel.discord.updatePresence()`. This also should not be closed as ClawHub work because the remaining request is for a missing runtime/plugin seam, which VISION.md explicitly treats as plugin API design work when a feature cannot be built as a plugin yet.
