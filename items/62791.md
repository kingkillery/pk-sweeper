## Summary

Keep open. Current `main` appears to have fixed the late stdout/stderr `Agent listener invoked outside active run` crash, but there is no strong evidence that the separate Node 24.14.0 AJV startup/validation crash is resolved. The code and docs on `main` still recommend Node 24 and still eagerly compile gateway/plugin AJV validators at startup, so the issue as filed is only partially addressed.

## Evidence

- **Late-output crash is explicitly guarded on main:** `src/agents/bash-tools.exec-runtime.ts:561` adds an `updatesDisabled` guard specifically to prevent the `"Agent listener invoked outside active run"` crash after a run settles, and `src/agents/bash-tools.exec.ts:1717` disables updates on abort before any late stdout/stderr can reach the disposed run.
