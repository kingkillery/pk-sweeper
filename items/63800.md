## Summary

Current `main` at `86f8c826e20a01d1bba67c0ba6e73182fbf0455f` does not implement offline APNs chat reply alerts. APNs support exists for explicit push and wake flows, but the chat completion paths still only broadcast websocket/session events, and the PR discussion points to ongoing related work rather than an obsolete request.
