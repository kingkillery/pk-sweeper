## Summary

Keep open. Current `main` has moved the default Codex OAuth path to `openai-codex/gpt-5.5`, but the codebase and live-test guidance still treat older Codex OAuth models such as `gpt-5.2` and `gpt-5.3-codex` as expected-to-work. The reporter also explicitly framed this as an OpenClaw-layer tracking/mitigation issue even if the rejection originates upstream, so there is not strong enough evidence for a conservative auto-close.
