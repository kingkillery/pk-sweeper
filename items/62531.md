## Summary

Current `main` already preserves the existing primary model when the configure wizard re-applies provider auth, with regression coverage and user-facing docs. The fix is on `main` via commit `8d57d745cf5f1d83648e3a755957825805f6d77b`, but it does not appear to be in a tagged release yet.
