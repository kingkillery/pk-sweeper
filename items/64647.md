## Summary

Keep open. Current `main` does not implement per-agent memory backend overrides, so this PR is not obsolete. The request is coherent, the intended config shape is still absent from types/schema/docs/runtime resolution on `main`, and the discussion indicates the design is useful with only a likely small typing fix remaining.
