## Summary

Keep open. Current `main` still has the reported promotion behavior: generic policy keys are in the shared single-account move list, channels without `namedAccountPromotionKeys` still return that full list when named accounts exist, and the doctor migration still moves those keys into `accounts.default`. Only Telegram and Matrix currently declare `namedAccountPromotionKeys` under `extensions/`, so the report remains applicable to most channels.
