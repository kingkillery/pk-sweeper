## Summary

Keep #33329 open. Current main has partial implementation: Copilot, Hugging Face, Ollama, and Bedrock now have plugin-owned discovery toggles, and Bedrock discovery is documented. But the issue asks to document toggles for all affected implicit discovery mechanisms, and the public/provider docs still do not document how to disable Copilot, Hugging Face, or Ollama discovery. The later comment also requests a global disable-all model discovery flag, which I did not find on current main. Related PR #33365 proposed the exact issue fix but is closed unmerged, so it is not a canonical superseding item.
