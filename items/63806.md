## Summary

Keep open. Current `main` still has the underlying `exec` workdir bug: `resolveWorkdir` checks the raw path with `statSync(workdir)` and silently falls back to cwd/home instead of expanding `~` or surfacing an invalid-path error. This PR appears directionally useful, but review feedback also identified unrelated `CHANGELOG.md` churn that should be cleaned up before merge.
