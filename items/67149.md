## Summary

Keep open. `origin/main` still uses the existing CDP screenshot path and does not contain the Hyprland fallback, so this PR is not implemented on main. The PR discussion also still contains substantive unresolved correctness concerns around tab targeting and compositor-scope capture, which makes conservative cleanup closure inappropriate.
