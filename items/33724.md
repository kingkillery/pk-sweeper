## Summary

Keep open. Current main still hard-codes config backup location, retention, and trigger behavior, so the requested user-configurable path/count/trigger capability remains missing. The related implementation PRs in the supplied GitHub context (#33896 and #34478) were closed without merge, and the cron workaround discussed there does not stop or reconfigure OpenClaw’s built-in `.bak` files.
