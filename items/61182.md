## Summary

Keep open. Current main has explicit cleanup and timeout guards in the memory-search path, but I did not find strong enough evidence that #61182 is already fixed on shipped code. The linked discussion points to a broader unresolved CLI hang class that was still being reported after this issue was filed, so closing would be premature.
