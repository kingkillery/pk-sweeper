## Summary

Keep open. Current main still has a hard default 2-minute reply typing TTL and does not expose a user-facing TTL config; only typing start mode and refresh cadence are configurable. The related PR #42851 is useful prior work but remains open/unmerged, so the requested capability is still missing on main.
