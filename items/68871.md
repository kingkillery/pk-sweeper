## Summary

Keep open. Current `main` still enables env-proxy routing whenever proxy env vars are present and only falls back to direct fetch when the proxy config is missing or malformed; there is no reachability probe or automatic retry-to-direct path when a configured proxy listener is down.

## Evidence
