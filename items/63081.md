## Summary

Keep open. Current `main` still has the exact unguarded `pending ?? {}` / `paired ?? {}` load path in both `src/infra/device-pairing.ts` and `src/infra/node-pairing.ts`, while `src/infra/json-files.ts` will return a parsed JSON array unchanged. That means the PR’s reported `[]` corruption path is still credible on `main`, and I did not find a regression test that locks in array-to-object recovery.
