## Summary

Keep open. Current `main` still hard-depends on `node:sqlite`, treats missing FTS5 as a degraded runtime state instead of fixing or documenting it, and `openclaw doctor` still only probes embedding readiness rather than FTS availability. The reported keyword-search degradation remains real on code inspection.
