## Summary

Keep issue #28965 open. Current main has useful partial behavior for model config and per-session overrides, but it does not implement the requested official `openclaw models reconcile` command or a bulk `openclaw sessions model-reset` CLI with agent/all-agents/store/match/dry-run/json support. The issue is authorAssociation `NONE` and only labeled `enhancement`, so there is no protected maintainer/label reason to keep it open, but meaningful requested core work remains.
