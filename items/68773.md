## Summary

Keep open. Current `main` still contains the exact `?? {}` fallback in `readActiveMemorySearchDebug`, which makes payload-less newer `memory_search` results truthy and able to shadow older entries with real debug data. The proposed PR describes a real bug and current tests on `main` do not cover that regression path.

## Evidence
