## Summary

Keep open. Current `main` does not already carry the PR's `.dockerignore` reductions, and the latest PR shape is not safe to close around because excluding `test/` would break active repo-root E2E Docker builds that still `COPY test ./test`. Recent `main` work improved Docker dependency-context handling in a different way, but it does not supersede this PR.
