## Summary

Keep open. Current `main` still does not register a `--model` option for `openclaw agent`, while the CLI-backends docs still advertise `openclaw agent --message "hi" --model codex-cli/gpt-5.5`. The mismatch also exists in the latest release (`v2026.4.23`), so this is neither fixed on `main` nor obsolete.
