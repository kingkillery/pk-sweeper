## Summary

Keep PR #61089 open. Current main still has the browser-reachable slash-command import path described by the PR: `ui/src/ui/chat/slash-commands.ts` imports `commands-registry.shared.ts`, which imports `thinking.js`, and `thinking.js` imports provider runtime. The PR still contains useful remaining work, but maintainer discussion says it should be narrowed/rebased because the session chat-type part is now obsolete on main.
