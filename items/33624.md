## Summary

Keep #33624 open. Current main has a documented WebSocket Gateway protocol and a TypeBox-to-JSON-Schema generator, but it does not expose or publish a Swagger/OpenAPI schema for programmatic clients. The requested capability remains meaningful core/API work rather than an auto-close candidate.
