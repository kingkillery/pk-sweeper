## Summary

Keep PR #60595 open. Current main still returns `structuredClone(store)` unconditionally from `loadSessionStore`, so the requested `skipCache: true` fast path is not implemented. The related PRs in the supplied context are closed and unmerged, while #60595 appears to be the clean two-file successor rather than something superseded by a merged canonical fix.
