## Summary

Keep open. Current `main` still does not implement `filter`, `sort`, `field_names`, or `automatic_fields` for `feishu_bitable_list_records`, and the PR discussion contains unresolved correctness concerns about the proposed `sort` schema guidance and missing validation.
