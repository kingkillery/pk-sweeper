## Summary

Keep open. Current `main` appears to have improved the ACP announce path by moving completion context into `internalEvents` that are prepended to the user/body prompt instead of mutating the system prompt, but the item is not fully resolved: old-image pruning still intentionally rewrites earlier transcript messages, and I did not find a regression proving byte-identical heartbeat/chat system prompts for the same session.
