## Summary

Keep open. Current main added public `llm_input`/`llm_output` hook seams around Pi LLM calls, which addresses part of the requested hook-point gap, but the bundled OpenTelemetry path still exports custom `openclaw.*` spans/metrics over OTLP/HTTP only and I found no `gen_ai.*`/OpenLLMetry/IITM integration on `main`.
