## Summary

Keep PR #61100 open. Current main still has the reported Discord discoverability gap: Discord message tool discovery does not advertise `upload-file`, and the Discord runtime does not include a dedicated `uploadFile` action dispatch path. The PR remains a relevant candidate fix for #60652 rather than an item that can be conservatively closed.
