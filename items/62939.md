## Summary

Keep open. Current `main` has related prompt-injection hardening for untrusted metadata and external-content tool outputs, but it does not implement the issue's requested cross-cutting boundary defense at generic message/tool-result transport boundaries. The feature request is coherent, core-scope security work, and the issue already has a linked open PR (#62973), so conservative cleanup should not auto-close it.
