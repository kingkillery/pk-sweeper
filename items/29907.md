## Summary

Keep open. Current main still lacks a non-LLM cron payload kind for direct exec/Lobster execution with conditional agent handoff; the cron contract and runtime only support main-session system events or agent turns. This is not a conservative clawhub close because the requested capability needs a core cron/protocol/plugin extension point, not just packaging a workflow.
