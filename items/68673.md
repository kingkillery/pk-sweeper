## Summary

Keep open. Current `main` still has the `update_plan` no-progress hashing gap this PR targets, but the PR head over-collapses `update_plan` shape by discarding step text. That creates a plausible regression where materially different plans with the same status pattern hash as identical and can trip the 30-call global circuit breaker, so this is not a safe conservative close.
