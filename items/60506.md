## Summary

Keep open. Current main does not implement the PR’s main requested capability: selectable derived model refs using `agents.defaults.models[*].routeTo`, nor per-entry `contextTokens` in that same config surface. The existing `contextTokens` support found on main is for other model/provider or agent-level config surfaces, not the `AgentModelEntryConfig` used by `agents.defaults.models`. The PR also has unresolved review feedback about async provider preparation for routed cross-provider refs, so it is not a clean duplicate or already-implemented cleanup candidate.
