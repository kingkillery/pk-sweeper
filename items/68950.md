## Summary

Keep open. Current `main` still hard-requires `plugins.entries.memory-lancedb.config.embedding.apiKey` and wires embeddings through a raw OpenAI API key. The docs also explicitly say Codex OAuth does not satisfy embedding requests, so the reported OAuth-only limitation still matches shipped/current behavior.
