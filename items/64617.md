## Summary

Keep open. Current `main` already avoids flagging archived `*.jsonl.deleted.<timestamp>` files in production, but it does not contain the doctor-level regression test proposed here. Existing coverage is helper-level only, so this PR still adds concrete value and is not closeable under the conservative cleanup reasons.
