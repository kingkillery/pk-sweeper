## Summary

Keep open. Current main has generic multi-profile auth rotation, but it does not implement the requested OpenRouter-native multi-key load balancing/fallback well enough to close this issue. OpenRouter profiles are explicitly exempted from cooldown/failure persistence, so repeated 429/quota failures are not tracked across runs.

## Evidence
