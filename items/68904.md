## Summary

Keep open. Current `main` still maps the Dreaming Advanced "From Daily Log" summary/origin/grouping to `groundedCount` rather than `dailyCount`, matching the reporter's claim. The issue is recent, reproducible from source, and already has a referenced open PR (#68942), so there is no conservative-close basis.
