## Summary

Keep open. Current main has some related model override source plumbing, but this draft PR is not implemented or cleanly superseded: several requested guardrails are still absent from main, including the Discord premature-disconnect guard, unique bound-channel delivery fallback, separated health probe/snapshot refreshes, and relaxed macOS gateway connect budgets.
