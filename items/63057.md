## Summary

Keep open: current `main` still contains the reported `|| 1024` fallback in Anthropic thinking params, and `adjustMaxTokensForThinking` can legitimately resolve the budget to `0`, so the PR addresses a live bug. The only clear gap from the discussion is missing regression coverage for the zero-budget and default-fallback paths.
