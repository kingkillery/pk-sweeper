## Summary

Current `main` does not implement the proposed plugin runtime API (`api.runtime.subagent.spawnDetached`), so this is not `implemented_on_main`, and the PR still appears coherent and potentially useful because core detached-subagent machinery already exists underneath and the PR discussion shows the main review concerns were addressed on the branch.
