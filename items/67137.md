## Summary

Keep open. Current `main` still contains the unsafe `.trim()` call sites this PR targets, and the repo still lacks the undefined-response regression coverage proposed here. Related discussion also shows that the obvious `?? ""` fallback approach needs maintainer judgment because it can mask required-input failures by turning them into blank values.
