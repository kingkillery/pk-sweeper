## Summary

Keep open. Current main still has the ASCII-only Discord mention candidate regex and account-scoped directory lookup, so the observable CJK bot-to-bot mention rewriting request is not implemented on main. The PR discussion also contains unresolved correctness concerns around cross-account fallback ambiguity, guild-scoped nicknames, and broader mention boundaries, so it should not be conservatively closed as implemented or obsolete.
