## Summary

Keep open. Current `main` does not implement this PR's API/runtime behavior: `ContextEngineFactory` is still no-arg, `resolveContextEngine()` still accepts only `config`, run/compaction/subagent paths do not pass runtime context, and the public context-engine docs still show the no-arg factory form. The PR also has recent maintainer follow-up in the discussion, so this is not conservative cleanup material.
