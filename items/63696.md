## Summary

Keep open. Current `main` still has the retry path that preserves the original user body on fallback attempts, but it does not have the PR’s run-local suppression plumbing for transcript persistence. That leaves the duplicate-user-persistence bug plausible, and this PR remains a scoped, useful fix rather than a cleanup candidate.
