## Summary

Current `main` does not implement `channels.feishu.customEventHandlers`, and the documented plugin-hook surface still does not expose raw Feishu event subscriptions. The PR remains a concrete, reviewable feature proposal rather than an obsolete or ClawHub-only item, so there is not strong evidence for any auto-close path.
