## Summary

Keep open. Current `main` still treats `skills.load.extraDirs` as explicit skill roots, with only a narrow heuristic for `<dir>/skills/<name>/SKILL.md`. It does not support pointing one `extraDirs` entry at a parent plugin/marketplace directory and discovering deeper layouts like `plugins/<name>/skills/<name>/SKILL.md`, and there is no glob or recursive config surface documented.

## Evidence

- **Config contract is still plain directory list:** `SkillsLoadConfig.extraDirs` is documented as additional skill folders to scan, and each directory should contain skill subfolders with `SKILL.md`; there is no recursive/glob/scan-depth option in the type surface.
  - file: [src/config/types.skills.ts:10](https://github.com/openclaw/openclaw/blob/ec8dbc4595583816f71af2568df61dbe7c090e1f/src/config/types.skills.ts#L10)
  - command: `nl -ba src/config/types.skills.ts | sed -n '1,200p'`
  - sha: [ec8dbc459558](https://github.com/openclaw/openclaw/commit/ec8dbc4595583816f71af2568df61dbe7c090e1f)
- **Loader only scans immediate children after optional `dir/skills` unwrap:** `resolveNestedSkillsRoot()` only rewrites `dir` to `dir/skills` when it finds `dir/skills/*/SKILL.md`. After that, `loadSkillEntries()` iterates immediate child directories and checks only `<child>/SKILL.md`. That still misses `plugins/<name>/skills/<name>/SKILL.md` when `extraDirs` points at `plugins/`.
  - file: [src/agents/skills/workspace.ts:323](https://github.com/openclaw/openclaw/blob/ec8dbc4595583816f71af2568df61dbe7c090e1f/src/agents/skills/workspace.ts#L323)

  - sha: [ec8dbc459558](https://github.com/openclaw/openclaw/commit/ec8dbc4595583816f71af2568df61dbe7c090e1f)
- **Watcher patterns are not recursive:** The skills watcher only watches `<root>/SKILL.md` and `<root>/*/SKILL.md` for each configured root. There is no `**` pattern or recursive expansion for deeper trees.
  - file: [src/agents/skills/refresh.ts:81](https://github.com/openclaw/openclaw/blob/ec8dbc4595583816f71af2568df61dbe7c090e1f/src/agents/skills/refresh.ts#L81)

  - file: [docs/tools/skills-config.md:13](https://github.com/openclaw/openclaw/blob/ec8dbc4595583816f71af2568df61dbe7c090e1f/docs/tools/skills-config.md#L13)
