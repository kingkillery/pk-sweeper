## Summary

Keep open. Current `main` still has two completion paths that both call `completeSubagentRun(..., triggerCleanup: true)` for the same run, and there is no browser-cleanup dispatch guard analogous to `endedHookEmittedAt`. The low-level browser tab close path is idempotent, but the wrapper-layer duplicate work described in the issue still matches the code on `main`.
