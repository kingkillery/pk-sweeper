## Summary

Keep open. Current `main` still lacks bundled OpenRouter audio transcription support, so this PR is not `implemented_on_main` or `cannot_reproduce`. There is already a ClawHub OpenRouter transcription plugin/skill and a first-class plugin seam for media-understanding providers, but that is a workaround/alternative path, not evidence that the bundled OpenRouter provider change is obsolete.
