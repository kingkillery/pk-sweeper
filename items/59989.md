## Summary

Keep PR #59989 open. Current main still unsubscribes immediately after aborting in-flight compaction, so the PR’s underlying event-loss concern is not implemented on main. The PR also has unresolved review feedback about async unsubscribe semantics and retry handling, so it is not safe to close as implemented or duplicate from the available evidence.
