## Summary

Keep open. Current `main` does not implement OAuth for remote MCP servers: the MCP config/types/docs only support static headers, and the transport/runtime code does not thread an OAuth auth provider or retry unauthorized remote MCP connections. The PR discussion also surfaced unresolved correctness issues in the proposed implementation, so there is not strong evidence for a conservative close.

## Evidence
