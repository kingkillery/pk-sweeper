## Summary

Current `main` does not implement inbound tapback/reaction forwarding for the legacy `imessage` channel. The checked-in code only parses ordinary message payloads and routes `message` notifications into text dispatch, while docs explicitly position `imessage` as a legacy path and BlueBubbles as the iMessage integration with reaction support.
