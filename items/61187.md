## Summary

Keep PR #61187 open. Current main does not implement the requested model-picker fix for Kimi, and the PR still contains useful work plus follow-up fixes for the review comments. Moonshot has a newer static catalog path on main, but the gateway/UI model catalog path still depends on provider `augmentModelCatalog` hooks for supplemental picker rows, and neither Kimi nor Moonshot currently defines that hook on main.
