## Summary

Keep open. Current main still falls back to config-only status for every `callGateway` failure in `channelsStatusCommand`, including `probe: true`, and the PR's focused regression test is not present. The gateway handler still awaits plugin probe hooks directly, so a probe hook failure can still become the command-side failure path this PR addresses.
