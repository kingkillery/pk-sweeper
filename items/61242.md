## Summary

Keep open. Current `main` at `59523e66da03591bc7bcc183a11f690fa0aae641` does not contain the PR's managed child-task Task Flow UX work: no `tasks flow retry` CLI, no managed-child retry/runtime API, no gateway/web control surface, no managed-flow auto-close-on-child-completion path, and `sessions_spawn` still lacks the documented `lane`/`taskFlow` inputs.
