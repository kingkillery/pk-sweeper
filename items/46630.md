## Summary

Current `main` has slash-command discoverability in WebChat, but it does not implement the requested configurable shortcut/keyword sidebar panel. The request remains an in-scope WebChat/UI enhancement rather than a ClawHub-only idea.

## Evidence

- **WebChat config/docs do not expose shortcut sidebar settings:** `docs/web/webchat.md:58`-`75` lists the current WebChat-specific configuration surface and only documents `gateway.webchat.chatHistoryMaxChars`; there is no `webchat.shortcuts` or separate shortcuts file support.
  - file: [docs/web/webchat.md:58](https://github.com/openclaw/openclaw/blob/56fe2aab9c87596515f2280c8060dc740ea64a88/docs/web/webchat.md#L58)
  - sha: [56fe2aab9c87](https://github.com/openclaw/openclaw/commit/56fe2aab9c87596515f2280c8060dc740ea64a88)
- **Current chat UI uses a slash-command menu, not a persistent shortcut sidebar:** `ui/src/ui/views/chat.ts:502` advertises `/` for commands, and `ui/src/ui/views/chat.ts:625`-`744` renders a transient grouped slash menu with descriptions and click selection. That overlaps with command discoverability, but it is not the requested collapsible sidebar panel listing user-defined prompt shortcuts.
  - file: [ui/src/ui/views/chat.ts:625](https://github.com/openclaw/openclaw/blob/56fe2aab9c87596515f2280c8060dc740ea64a88/ui/src/ui/views/chat.ts#L625)
  - sha: [56fe2aab9c87](https://github.com/openclaw/openclaw/commit/56fe2aab9c87596515f2280c8060dc740ea64a88)
- **Runtime discoverability is command-driven and sourced from `commands.list`:** `ui/src/ui/chat/slash-commands.ts:431`-`459` refreshes the menu from Gateway `commands.list`, and `ui/src/ui/chat/slash-commands.node.test.ts:114`-`170` verifies dock/plugin/skill commands appear. This supports existing command discovery, but not arbitrary config-defined keyword prompts with labels/descriptions/icons/categories.
  - file: [ui/src/ui/chat/slash-commands.ts:431](https://github.com/openclaw/openclaw/blob/56fe2aab9c87596515f2280c8060dc740ea64a88/ui/src/ui/chat/slash-commands.ts#L431)
  - sha: [56fe2aab9c87](https://github.com/openclaw/openclaw/commit/56fe2aab9c87596515f2280c8060dc740ea64a88)
