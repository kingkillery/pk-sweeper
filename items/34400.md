## Summary

Keep open. Current main partially supports recursive collection of built-in memory files, but the item’s follow-up runtime and user-facing issues are not fully fixed on main: dated subdirectory memory files are still not recognized for temporal decay, dirty search sync is still intentionally fire-and-forget, QMD watching still builds glob watch paths under chokidar 5, and several user-facing strings still describe only memory/*.md.
