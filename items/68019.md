## Summary

Keep #68019 open. Current main still uses a single fixed-size `chunkMarkdown` path with only `tokens` and `overlap` config; it does not have a chunking strategy API, strategy selection, or per-strategy reindex metadata. The PR is an unmerged, focused memory-core feature with tests and review follow-up, so no conservative close reason applies.
