## Summary

Keep open. Current `main` has partial overlap for local OpenAI-compatible backends, but it does not fully implement this PR's generic-local compat path or inline text tool-call execution. The code still excludes `endpointClass: "local"` from native streaming-usage compat, still skips the boundary-aware transport whenever a non-default `currentStreamFn` is already set, and still treats `choice.delta.content` as plain text rather than upgrading inline tool-call payloads into structured tool calls.

## Evidence
