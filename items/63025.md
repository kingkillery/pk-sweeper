## Summary

Keep open. Current `main` still has the zero-output completion path: after dispatch it records `completed` and returns `queuedFinal`/`counts` without any fallback when `replies`, tool outputs, and block outputs are all empty. The PR’s proposed direct/private-chat fallback and its regression tests are not present on `main`, and the PR discussion shows the author already addressed the main review concern by narrowing the behavior away from channel ack cases.
