## Summary

Current main, and the released v2026.4.23 tree, already wire `config.env` proxy vars into the runtime env and bootstrap Undici's `EnvHttpProxyAgent` before `memory-core` remote embedding requests run.

## Evidence

- **`openclaw.json` env vars are applied before runtime config resolution:** `resolveConfigForRead` applies `config.env` into the runtime env before substitution, so `HTTP_PROXY` / `HTTPS_PROXY` from `openclaw.json` become process env for the CLI/runtime.
  - file: [src/config/io.ts:1054](https://github.com/openclaw/openclaw/blob/a9797214338ba31c52c796adbb75afb16e0684a9/src/config/io.ts#L1054)
  - sha: [a9797214338b](https://github.com/openclaw/openclaw/commit/a9797214338ba31c52c796adbb75afb16e0684a9)
- **CLI bootstraps env-proxy support:** `runCli` calls `ensureGlobalUndiciEnvProxyDispatcher()` during startup, before command execution.
  - file: [src/cli/run-main.ts:187](https://github.com/openclaw/openclaw/blob/a9797214338ba31c52c796adbb75afb16e0684a9/src/cli/run-main.ts#L187)
  - sha: [a9797214338b](https://github.com/openclaw/openclaw/commit/a9797214338ba31c52c796adbb75afb16e0684a9)
- **Undici installs `EnvHttpProxyAgent` from proxy env:** `ensureGlobalUndiciEnvProxyDispatcher` checks for env proxy configuration and swaps the global dispatcher to `EnvHttpProxyAgent`, which is the standard `HTTP_PROXY` / `HTTPS_PROXY` path.
  - file: [src/infra/net/undici-global-dispatcher.ts:84](https://github.com/openclaw/openclaw/blob/a9797214338ba31c52c796adbb75afb16e0684a9/src/infra/net/undici-global-dispatcher.ts#L84)
  - sha: [a9797214338b](https://github.com/openclaw/openclaw/commit/a9797214338ba31c52c796adbb75afb16e0684a9)
- **Memory remote embedding HTTP uses the shared fetch path:** Remote embedding requests go through `withRemoteHttpResponse`, which delegates to the shared guarded fetch layer rather than a separate direct client.
  - file: [src/memory-host-sdk/host/remote-http.ts:14](https://github.com/openclaw/openclaw/blob/a9797214338ba31c52c796adbb75afb16e0684a9/src/memory-host-sdk/host/remote-http.ts#L14)
  - sha: [a9797214338b](https://github.com/openclaw/openclaw/commit/a9797214338ba31c52c796adbb75afb16e0684a9)
- **Regression coverage exists for env proxy bootstrap:** Tests assert that env proxy configuration installs `EnvHttpProxyAgent` on the global dispatcher.
  - file: [src/infra/net/undici-global-dispatcher.test.ts:207](https://github.com/openclaw/openclaw/blob/245451b6a9c529464e177afa65013adca0d63eb5/src/infra/net/undici-global-dispatcher.test.ts#L207)
  - sha: [245451b6a9c5](https://github.com/openclaw/openclaw/commit/245451b6a9c529464e177afa65013adca0d63eb5)
- **Regression coverage exists for config env injection:** Tests assert that config `env` values are applied when missing from the process environment.
  - file: [src/config/config.env-vars.test.ts:17](https://github.com/openclaw/openclaw/blob/245451b6a9c529464e177afa65013adca0d63eb5/src/config/config.env-vars.test.ts#L17)
  - sha: [245451b6a9c5](https://github.com/openclaw/openclaw/commit/245451b6a9c529464e177afa65013adca0d63eb5)
