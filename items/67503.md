## Summary

Keep #67503 open. Current main still includes the truncated 160-character inbound message preview in Slack, MS Teams, and Mattermost system-event text, and downstream system-event formatting still exposes that text as `System:` lines. The linked fix PR #67761 is open/unmerged in the provided context, so it has not superseded the bug as shipped behavior.

## Best Possible Solution

Keep the issue open until #67761 or an equivalent maintainer-approved patch lands. The fix should remove message body previews from Slack, MS Teams, and Mattermost inbound system-event labels while preserving full body delivery through the normal inbound envelope/context, and should add focused regression tests for the system-event text in each affected channel.
