## Summary

Keep open. Current `main` still uses the legacy non-`all` DNS callback shape in `createPinnedLookup`, and the tests still codify that behavior. `main` does include a separate workaround for the Node 25 `globalThis.fetch` vs undici-dispatcher incompatibility, so the broader fetch-failure story is only partially addressed, not fully implemented on `main`.
