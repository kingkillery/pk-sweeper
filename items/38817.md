## Summary

Current `main` at `d3dd61f2c5bd6cea175544f3a76a68ec11fa17c2` does not show a shipped Volcengine/Kimi fix for unsupported tool-schema keywords. The cleanup path is now generic, but it only activates when the resolved model carries `compat.unsupportedToolSchemaKeywords`; the Volcengine plugin/catalog on `main` does not attach that compat for its Kimi routes, so the reported `minLength` failure still looks plausible rather than implemented.
