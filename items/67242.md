## Summary

Keep open. Current `main` does not contain the requested `openclaw/plugin-sdk/embeddings` subpath: package exports, the SDK entrypoint registry, docs, and bundled consumers still point at `plugin-sdk/memory-core-host-engine-embeddings`. The existing old subpath is already a public re-export of the embeddings API, so this PR is additive cleanup rather than an already-landed fix, and there is no strong auto-close basis.
