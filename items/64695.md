## Summary

Keep open. Current `main` still hard-requires `node:sqlite` for the task registry and does not contain the linked JSON fallback, so the reported runtime incompatibility is not implemented away. The crash symptom may be reduced on current `main` because scheduled sweep failures are explicitly caught, but the underlying unsupported-runtime failure remains plausible.
