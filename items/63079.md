## Summary

Current `main` improves Google Chat diagnostics around add-on principal mismatches, but it still requires `channels.googlechat.appPrincipal` for `audienceType: "app-url"` and the current setup/docs surfaces still do not expose that requirement clearly. The issue's user-facing failure mode therefore remains plausible on current `main`, so this is not strong enough to close conservatively.
