## Summary

Current `main` does inject `web_search` into OpenAI-compatible chat-completions payloads and now has partial mitigations for strict local backends, but it still only recognizes tool calls from structured `tool_calls`. Backends like llama-server that emit tool-call JSON as plain assistant text remain unsupported, so the broader failure mode in #62692 is still plausible on `main`.
