## Summary

Keep open: current `main` still uses `pending ?? {}` / `paired ?? {}` in both pairing `loadState` paths, and `readJsonFile` still returns `JSON.parse(raw) as T` without runtime shape validation, so the reported `[]`-typed state-file path is not shown fixed on `main`.
