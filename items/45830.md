## Summary

Keep open. Current `main` still lacks the origin-delivery snapshot fields and fallback logic this PR adds, while isolated cron sessions on `main` still clear `lastChannel`/`lastTo`/`lastAccountId`. The reported bug remains plausible on current code, and the PR discussion shows the contributor addressed the main review follow-ups rather than making the branch obsolete.
