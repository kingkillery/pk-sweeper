## Summary

Keep open. Current `main` still uses strict realpath containment for skill roots, which would reject symlinked skill directories under `~/.agents/skills` and `<workspace>/.agents/skills`. The PR appears to address a real gap on current `main` rather than duplicating an implementation already shipped there.
