## Summary

Keep open. Current `main` and the latest tagged release still implement `plugins.allow_phantom_entries` by comparing against `listChannelPlugins()`, while that API only returns runtime-loaded channel plugins. The repo docs explicitly tell users to include bundled plugins like `browser` in `plugins.allow`, so the reported false positive remains plausible and is not implemented on `main`.
