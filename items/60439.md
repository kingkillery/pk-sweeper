## Summary

Keep PR #60439 open. Current main has built-in `openclaw update` plugin sync and `--no-restart` support, but it does not contain this PR's standalone phased update script, and the PR discussion still contains substantive safety-review feedback rather than a clear canonical superseding item. The related maintainer issue #64892 is relevant to update safety but tracks a different self-update bypass problem, not this PR's proposed gateway-plus-plugin phased script.
