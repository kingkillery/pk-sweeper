## Summary

Keep open. Current `main` still uses the strict MCP SDK `client.listTools()` path on `@modelcontextprotocol/sdk` 1.29.0, so this PR addresses a path that is not implemented on `main`. The downstream schema normalization in the agent pipeline happens after bundle MCP tool discovery, so it does not prove #63602 is already fixed. The PR discussion also surfaced a still-relevant maintainer judgment call around explicit non-object schemas, so conservative cleanup should not auto-close this PR.
