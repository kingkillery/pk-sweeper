## Summary

Keep open. Current `main` still does not implement the PR's opt-in send-media hydration seam, and the attached discussion shows the author already addressed the main regression concern by gating preloading rather than abandoning the change. This remains a plausible core/plugin-SDK improvement, so there is not strong evidence to auto-close it conservatively.
