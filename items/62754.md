## Summary

Keep open. Current `main` has a related fix for the same crash text in the exec-tool path and now bundles Pi `0.70.2`, but I did not find strong evidence that the specific embedded runner abort/timeout race from #62754 is fixed. If this rejection still escapes, the global unhandled-rejection handler would still treat it as fatal.
