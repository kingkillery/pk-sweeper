## Summary

Current `main` still drops embedded `turn_start`/`turn_end` events in the subscription dispatch layer, while repo docs say those events are part of the embedded session contract. This PR is not obsolete and still targets a real gap, so it should stay open for maintainer review rather than cleanup-closing.
