## Summary

Current `main` does not implement the PR’s native plugin MCP registration seam. The checked-out code still lacks `api.registerMcpServer(...)`, lacks manifest/registry support for plugin-owned MCP server registrations, and only reports MCP support for bundle-format plugins. This PR remains additive work rather than something already landed on `main`.
