## Summary

Keep open. Current `main` still drops `thinkLevel` and `verboseLevel` when mixed-content messages trigger inline-directive clearing, so this PR appears to address a live bug rather than something already implemented on `main`.

There is no preservation/reinjection of think or verbose levels here.
