## Summary

Keep open. Current `main` does not implement the PR's core behavior: loop detection is still disabled by default, `generic_repeat` is still warn-only until the global breaker, and exec approval resolution still uses `maxAsk(...)`. The shipped docs/tests/changelog also show those behaviors are currently intentional rather than already-fixed regressions.
