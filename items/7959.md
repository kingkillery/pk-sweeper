## Summary

Keep open. Current `main` bootstraps and reads workspace identity files and exposes `openclaw agents set-identity`, but it does not implement built-in identity-file version history, changelog diffs, or rollback. The only persisted workspace metadata is bootstrap/setup timestamps, and the docs still recommend using a private git repo for backup/recovery.

## Evidence

- **Workspace identity files are first-class bootstrap files:** The workspace runtime defines `AGENTS.md`, `SOUL.md`, `TOOLS.md`, `IDENTITY.md`, `USER.md`, `HEARTBEAT.md`, and `MEMORY.md` as bootstrap files, so this issue targets real core workspace surfaces.
  - file: [src/agents/workspace.ts:30](https://github.com/openclaw/openclaw/blob/cec34821754ad935c780805f3116f99449d46ac2/src/agents/workspace.ts#L30)
  - sha: [cec34821754a](https://github.com/openclaw/openclaw/commit/cec34821754ad935c780805f3116f99449d46ac2)
- **Workspace state stores only setup markers, not file history:** `WorkspaceSetupState` contains only `bootstrapSeededAt` and `setupCompletedAt`, and `ensureAgentWorkspace()` updates only those markers. There is no per-file revision log, diff storage, or rollback state.
  - file: [src/agents/workspace.ts:165](https://github.com/openclaw/openclaw/blob/cec34821754ad935c780805f3116f99449d46ac2/src/agents/workspace.ts#L165)
  - sha: [cec34821754a](https://github.com/openclaw/openclaw/commit/cec34821754ad935c780805f3116f99449d46ac2)
- **Tests confirm the persisted workspace state shape is timestamps only:** `readWorkspaceState()` in the workspace tests expects just `version`, `bootstrapSeededAt`, and `setupCompletedAt`, matching the implementation and showing no identity changelog/history payload exists.
  - file: [src/agents/workspace.test.ts:34](https://github.com/openclaw/openclaw/blob/cec34821754ad935c780805f3116f99449d46ac2/src/agents/workspace.test.ts#L34)
  - sha: [cec34821754a](https://github.com/openclaw/openclaw/commit/cec34821754ad935c780805f3116f99449d46ac2)

  - file: [docs/cli/agents.md:18](https://github.com/openclaw/openclaw/blob/cec34821754ad935c780805f3116f99449d46ac2/docs/cli/agents.md#L18)
