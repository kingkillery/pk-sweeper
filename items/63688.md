## Summary

Keep open. Current `main` does not implement this PR's core change for non-OpenAI/non-strict-agentic providers: the clean empty-response retry is still gated to the strict-agentic GPT-5 lane, while the newer model-agnostic retry on `main` only covers `stopReason="error"` silent failures. The PR also has unresolved review feedback for a likely orchestration regression on `yieldDetected`/`clientToolCall` turns plus style-policy issues in the added test file.
