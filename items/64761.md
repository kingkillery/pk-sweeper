## Summary

Current `main` still rebuilds and passes the full effective tool set, including each tool's full `description` and `parameters`, into embedded agent session creation on every run. Repo docs and context-report code still account for full tool-schema JSON as prompt/context cost, and the only documented mitigation I found is disabling tool support entirely with `compat.supportsTools: false`, not compacting or adaptively injecting schemas.
