## Summary

Keep open. Current main has partial theming support: Claw/Knot/Dash theme families, system/light/dark modes, and a browser-local custom theme slot imported from tweakcn. It does not implement the issue's requested six preset themes or built-in Custom Theme Studio with hue generation, gradient toggles, advanced color groups, and per-variable UI overrides. The recent discussion also still asks for proper built-in theming rather than one-off re-skinning.
