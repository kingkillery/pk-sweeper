## Summary

Keep open. Current main still does not expose a `tools` parameter on `sessions_spawn`; sub-agent tool policy is still configured through global/per-agent config, not per spawn call. This is a core session-tool and security-policy surface, so it is not a ClawHub-only cleanup item.

## Evidence
