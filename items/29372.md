## Summary

Keep open. Current main fixes the related cross-agent workspace inheritance case from #40171, but it does not implement the requested per-agent `agents.list[].subagents.workspace` override for default/same-agent subagents. Same-agent/default subagent spawns still intentionally preserve the caller's inherited workspace, and the config schema/docs expose `subagents.allowAgents`/`requireAgentId` but no `workspace` key.
