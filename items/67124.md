## Summary

Keep open. Current `main` still requires pairing for backend scope upgrades, and `main` now includes a regression test that explicitly expects a local backend reconnect with broader scopes to fail with `pairing required`. The PR head also still matches the ineffective guard called out in review, so there is not strong evidence for any close reason.
