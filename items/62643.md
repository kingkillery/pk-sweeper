## Summary

Current `main` has the `purpose?: "default" | "status"` type and already passes `purpose: "status"` through the status scan, but it still lacks the PR‚Äôs actual additions: the public contract JSDoc for third-party memory plugins, the clarifying probe comment, and the dedicated `resolveSharedMemoryStatusSnapshot` tests. This PR is still additive, and the discussion ties it to a real third-party plugin false-negative (`openclaw-memory-openbrain`), so there is not strong evidence to close it.
