## Summary

Keep #60875 open. Current main has a separate workspace discovery hardening that confines auto-discovery to `.openclaw/extensions`, but it does not implement this PR's typed manifest failure reason or workspace-only downgrade from error to warning. The narrower case described by the PR can still matter for loose `.js` files or directories with `index.js` under the workspace extensions root.
