## Summary

Keep open. Current `main` does not already implement the PR's first two fixes: auto-failover still persists into session override fields, and there is still no separate context-engine conversation key to distinguish `/new` from `/reset`. The PR also has unresolved review feedback about the new conversation-key wiring, so this is not a conservative auto-close candidate.
