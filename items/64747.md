## Summary

Keep open. Current `main` does not include this PR's gateway-boundary change: `src/gateway/server.ts` still lazy-loads `server.impl.js` without calling `ensureGlobalUndiciEnvProxyDispatcher()`, and the added regression test file from the PR is absent. `main` does now install the env proxy dispatcher in CLI startup (`src/cli/run-main.ts`), which may cover some `openclaw gateway` launches, but that is not the same implementation and does not make this PR clearly obsolete for non-CLI or direct `startGatewayServer()` entrypoints.
