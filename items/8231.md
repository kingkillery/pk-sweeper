- The shipped solution is a managed `web_search` provider plus `openclaw infer` CLI, not a separately named new core "skill"; DuckDuckGo is also documented as experimental/unofficial and may hit bot challenges.

## Close Comment

Closing this after a Codex review because current `main` already ships the requested capability. OpenClaw has a terminal web-search surface via `openclaw infer web search` (`src/cli/capability-cli.ts:1752`, `docs/cli/infer.md:91`), and the managed `web_search` feature now supports key-free providers, including DuckDuckGo (`docs/tools/web.md:63`, `docs/tools/web.md:178`, `docs/tools/duckduckgo-search.md:10`, `extensions/duckduckgo/src/ddg-search-provider.ts:36`). I also confirmed the same files are present in the shipped `v2026.4.22` release (`00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5`). This is not packaged as a separately named new core skill, but the requested outcome, terminal-based web search without requiring a Brave API key, is already implemented.
