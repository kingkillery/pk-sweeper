## Summary

Keep PR #60184 open. The linked bug is still present on current main: console capture writes every console level to the file logger without consulting `isFileLogLevelEnabled`, while docs say file logs are controlled by `logging.level`. The PR’s proposed fix still targets the implicated code path and does not appear implemented or superseded in the checked-out main history.
