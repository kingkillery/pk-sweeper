## Summary

Keep open. Current main partially supersedes the GPT-5.4 embedded-terminal fallback concern through the merged #70743 classifier work, but it does not implement this PR's generic raw provider `{ content: [] }` handling or the proposed Telegram typed error routing. The Greptile/Codex review comments were partly addressed in the PR branch, but those changes are not present on current main.
