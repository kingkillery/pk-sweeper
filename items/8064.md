## Summary

Keep open. Current `main` has partial setup improvements for skill onboarding, but the issue's main friction is still present: the bundled Notion skill still requires users to manually share an existing Notion page/database before it can do useful work, and there is no notion-specific bootstrap/OAuth/helper flow in the tree.
