## Summary

Keep open. Current `main` still has the stale subsystem child-logger cache that issue #62381 describes, and the PR adds a missing rollover regression test rather than duplicating an already-shipped fix. The PR is still relevant, though it also carries an unrelated subagent test change that should be reviewed/split separately.
