## Summary

Keep PR #60860 open. Current main has only partial google-vertex plumbing: env auth marker detection and endpoint classification exist, but the PR's requested separate Google Vertex provider, manifest registration, catalog, ADC token exchange, and Vertex-specific /projects/{project}/locations/{location}/... request routing are not present on main.
