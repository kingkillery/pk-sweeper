## Summary

The PR’s premise no longer matches current `main`: unset `tools.exec.host` now defaults to `auto`, not implicit `sandbox`, so the reported implicit-sandbox fail-closed gap is obsolete. Current code routes implicit `auto` to `gateway` when no sandbox runtime exists, while explicit `host=sandbox` still fails closed and the audit/docs match that behavior.
