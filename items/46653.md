## Summary

Keep open. Current `main` does not implement a persistent `openclaw tunnel` command, and the existing SSH support is narrower: temporary probe-time forwarding plus manual SSH/LaunchAgent documentation. This is not a ClawHub-only feature under `VISION.md`; it fits current core CLI ergonomics work. The PR may still be useful, but it is not already shipped and should not be closed as obsolete.
