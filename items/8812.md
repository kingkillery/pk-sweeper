## Summary

Current `main` does not implement auto-linkified URLs in tool output cards. Control UI tool outputs still render as plain text/code blocks, and sidebar formatting still wraps JSON as fenced code, so the feature request remains valid.

## Evidence

- **Expanded tool output stays raw text:** `ui/src/ui/chat/tool-cards.ts:353-370` renders expanded tool output as `<pre><code>${text}</code></pre>`, with no URL parsing or anchor generation. `ui/src/ui/chat/tool-cards.ts:500-507` uses that block for normal tool output cards.
