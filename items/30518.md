## Summary

Keep open. Current main has Telegram Bot API helper functions for pin/unpin and supports delivery-time pinning of newly sent Telegram messages, but the generic Telegram message action adapter still does not expose manual `pin` or `unpin` actions for existing messages, which is the issue's requested capability.
