## Summary

Keep open. Current `main` still has the exact failure path the report describes: `sessions_spawn` rejects `streamTo` for `runtime="subagent"`, the schema still gives no ACP-only description for `streamTo`, and the tool description still does not document runtime-specific parameter safety.
