## Summary

Keep PR #61483 open. Current main does not implement the proposed Android `http.request` node command, and the request is not cleanly closable as ClawHub/plugin scope because the remaining capability needs Android node runtime/protocol wiring plus gateway allowlist support, not just an installable plugin or skill.
