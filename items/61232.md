## Summary

Keep open. Current main does not implement the requested third-party Android Always/background location capability, and the open PR still contains meaningful unmerged work. Main still has only Off/WhileUsing location modes, migrates stored "always" back to WhileUsing, shows only Off/While Using in Settings, blocks background location.get whenever the app is not foreground, and lacks the background-location manifest/build flavor wiring proposed by the PR.
