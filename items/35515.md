## Summary

Keep open. Current main has an Android app and build path, but the requested official Android APK release is still not implemented: docs say the app has not been publicly released, the Android README still marks release hardening incomplete, the release script builds signed AAB bundles rather than APK release artifacts, and the latest GitHub release assets contain only macOS artifacts. The issue discussion has useful signals: a contributor offered help with release signing/secrets, AndroidForClaw was linked as a standalone workaround, and F-Droid was suggested as a distribution target, but none of those supersede the official OpenClaw APK request.
