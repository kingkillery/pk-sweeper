## Summary

Keep open. Current main still has the reported type/schema mismatch: runtime exec approval allowlist entries can carry source: "allow-always", but the gateway protocol schema for exec.approvals.set still omits source while using additionalProperties: false. The related PRtags group shows duplicate PRs (#60064 and #69575), but no canonical fix is merged in this checkout, and #60073 also contains a separate read-tool path-alias fix that is not superseded by the schema-only duplicates.
