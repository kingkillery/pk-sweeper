## Summary

Keep open. Current `main` does not implement the requested inactivity-based sub-agent stall recovery. The code and docs still expose only a wall-clock `runTimeoutSeconds` timeout, with no `stallNudgeSeconds`/`stallTimeoutSeconds` config, no sub-agent last-tool-call activity tracking, and no nudge-then-kill lifecycle.
