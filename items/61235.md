## Summary

Keep open. Current `main` still has the reported model-precedence and model-picker behaviors, and the CLI backend docs only cover generic defaults/troubleshooting rather than the three concrete Claude CLI pitfalls described in #61235 or the follow-up `not in catalog` confusion from comments.

## Evidence

- **Troubleshooting section does not document the reported pitfalls:** `docs/gateway/cli-backends.md` troubleshooting currently covers only CLI path, model aliasing, session continuity, and images. It does not mention `agents.list[].model` overriding `agents.defaults.model`, `input: "stdin"` vs `arg`, or Claude stream-json parsing/setup pitfalls.
