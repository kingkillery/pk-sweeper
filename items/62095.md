## Summary

Keep open. Current `main` still keeps the reported subagent completion-announce path on a 120s default timeout and explicitly retries gateway-timeout failures four times, so at least one core regression described in #62095 remains present. Some other sub-items appear partially addressed on newer releases (doctor legacy-key migrations exist now; private-LAN plaintext WS is now documented as an explicit env-gated path; stale-socket health logic was changed in the changelog), but the aggregate issue is not cleanly resolved end-to-end.
