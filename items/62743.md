## Summary

Keep open. Current `main` still has an unresolved approval-flow mismatch: the Control UI always shows an `Always allow` button, but exec policy can forbid `allow-always` when `exec.ask=always`, and the gateway still rejects that decision. That matches the reported symptom of approving once but being prompted again, so there is not enough evidence to close this as fixed or unreproducible.
