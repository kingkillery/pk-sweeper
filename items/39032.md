## Summary

Current `main` has related commentary/final-answer leak hardening, but I did not find strong evidence that issue #39032 is fully fixed. The subagent announce path still falls back to raw `toolResult` text without checking `isError`/`is_error`, and existing tests cover normal tool-output fallback plus commentary suppression, not the reported tool-failure leak path.
