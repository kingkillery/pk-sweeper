## Summary

Current main does not implement a user-facing `/models test-fallback` command or equivalent fallback-chain verifier. OpenClaw has live auth probes and internal fallback runtime logic, but the shipped `/models` slash command and `openclaw models` CLI do not expose a way to intentionally skip the primary model, walk the configured fallback chain, and report which fallback actually answered.

## Evidence

- **Repo-wide search found no implementation:** `rg -n "test-fallback" src docs` found no command or docs implementation for this feature; the only match was unrelated test data in `src/infra/machine-name.test.ts`.
  - command: `rg -n "test-fallback" src docs`
  - sha: [cec34821754a](https://github.com/openclaw/openclaw/commit/cec34821754ad935c780805f3116f99449d46ac2)
- **`/models` slash command has no `test-fallback` action:** `ParsedModelsCommand` only defines `providers`, `list`, and `add`, and `parseModelsArgs()` dispatches only those actions. There is no parser branch for `test-fallback`.
  - file: [src/auto-reply/reply/commands-models.ts:46](https://github.com/openclaw/openclaw/blob/cec34821754ad935c780805f3116f99449d46ac2/src/auto-reply/reply/commands-models.ts#L46)
  - sha: [cec34821754a](https://github.com/openclaw/openclaw/commit/cec34821754ad935c780805f3116f99449d46ac2)
- **CLI has no fallback-test subcommand:** `registerModelsCli()` registers `list`, `status`, `set`, `set-image`, `aliases`, `fallbacks`, `image-fallbacks`, `scan`, and `auth`. There is no `test-fallback` CLI command or option.
  - file: [src/cli/models-cli.ts:28](https://github.com/openclaw/openclaw/blob/cec34821754ad935c780805f3116f99449d46ac2/src/cli/models-cli.ts#L28)

- **Docs only document auth probing, not fallback-chain testing:** The public models docs list `status`, `list`, `set`, and `scan`, and describe `models status --probe` as a live auth probe against configured provider profiles. They do not document a command that skips the primary model and verifies the configured fallback chain.
  - file: [docs/cli/models.md:21](https://github.com/openclaw/openclaw/blob/cec34821754ad935c780805f3116f99449d46ac2/docs/cli/models.md#L21)

  - file: [src/commands/models/list.probe.ts:149](https://github.com/openclaw/openclaw/blob/cec34821754ad935c780805f3116f99449d46ac2/src/commands/models/list.probe.ts#L149)
