## Summary

Current `main` has partial hardening around sandboxing, tool policy, exec approvals, and an optional workspace-only filesystem guard, but it does not implement the proposed data-centric security profile. The request is still core-scope, and repo docs/tests show several requested controls remain incomplete or explicitly planned rather than shipped.
