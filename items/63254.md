## Summary

Keep open. Current `main` appears to have fixed the Matrix restart-replay symptom, but the broader cron/session-injection half of the report is not convincingly resolved: cron still feeds the shared system-event queue via `contextKey`, and that queue still only suppresses consecutive duplicate text rather than idempotently deduping by `contextKey` across later replays.
