## Summary

Current `main` still contains the `Math.max(0, activeRuns - 1)` clamp in `onRunEnd()`, but the checked-in tests only cover init, normal start/end, and abort behavior. The PR’s current patch is a narrow test-only addition for the missing underflow cases, so there is not strong evidence to auto-close it as implemented, obsolete, or incoherent.
