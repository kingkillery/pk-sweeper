## Summary

Keep open. Current `main` still treats sub-agent announce suppression as prompt/output-driven only (`ANNOUNCE_SKIP` / `NO_REPLY`) and does not expose a framework-level config or native `sessions_spawn` parameter for suppressing native sub-agent completion announces.

## Evidence

- **Docs still describe native subagents as always announcing unless the child emits a magic token:** `docs/tools/subagents.md:10`, `docs/tools/subagents.md:82-83`, and `docs/tools/subagents.md:236-247` say sub-agents announce back to the requester chat, and suppression only happens when the child reply is exactly `ANNOUNCE_SKIP` or the silent token `NO_REPLY`.
  - file: [docs/tools/subagents.md:10](https://github.com/openclaw/openclaw/blob/c0a7b6a510758645c5267734b984f4885a605b67/docs/tools/subagents.md#L10)
  - command: `nl -ba docs/tools/subagents.md | sed -n '1,280p'`
  - sha: [c0a7b6a51075](https://github.com/openclaw/openclaw/commit/c0a7b6a510758645c5267734b984f4885a605b67)
- **Agent defaults schema has no announce policy/suppress flag:** `src/config/types.agent-defaults.ts:388-410` lists subagent defaults such as `allowAgents`, `maxConcurrent`, `runTimeoutSeconds`, and `announceTimeoutMs`, but there is no `announcePolicy`, `suppressAnnounce`, or equivalent config knob.
  - file: [src/config/types.agent-defaults.ts:388](https://github.com/openclaw/openclaw/blob/c0a7b6a510758645c5267734b984f4885a605b67/src/config/types.agent-defaults.ts#L388)
  - command: `nl -ba src/config/types.agent-defaults.ts | sed -n '388,430p'`
  - sha: [c0a7b6a51075](https://github.com/openclaw/openclaw/commit/c0a7b6a510758645c5267734b984f4885a605b67)
- **`sessions_spawn` lacks a native subagent announce suppression parameter:** `src/agents/tools/sessions-spawn-tool.ts:117-125` defines the accepted parameters and includes `streamTo` only for ACP. `src/agents/tools/sessions-spawn-tool.ts:229-233` explicitly rejects `streamTo` unless `runtime=acp`, so native `runtime="subagent"` still has no per-spawn routing/suppression control.
  - file: [src/agents/tools/sessions-spawn-tool.ts:117](https://github.com/openclaw/openclaw/blob/c0a7b6a510758645c5267734b984f4885a605b67/src/agents/tools/sessions-spawn-tool.ts#L117)
