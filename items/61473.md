## Summary

Keep PR #61473 open. Current main still appears to have the old Discord model-picker confirmation path: it dispatches the hidden /model command, waits, re-reads the already-resolved route, and reports a mismatch without any direct session-store fallback persist. The related PR #53287 is closed but unmerged in the provided context, so it does not supersede this work on main.
