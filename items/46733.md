## Summary

Keep open. Current `main` still intentionally removes the Anthropic `context-1m` beta when the auth token looks like Anthropic OAuth/setup-token auth (`sk-ant-oat*`), so the runtime behavior reported in #46733 is still present. The display side appears improved on `main` because configured `context1m` now resolves to a 1M context window in context/status paths, but that is only a partial fix and does not resolve the OAuth runtime limitation.
