## Summary

Current `main` does not contain the PR's opt-in ACP parent completion notify feature. The `sessions_spawn` ACP surface still exposes only `streamTo:"parent"`, docs still describe parent updates as system events, and the ACP parent relay still emits terminal completion/error via system events rather than the subagent announce flow. This PR should stay open.
