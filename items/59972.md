## Summary

Current `main` already implements the requested Control UI/WebChat behavior: streaming auto-follow now stops when the user scrolls away from the bottom, new content is tracked below the viewport, and the UI exposes a `New messages` jump control. The implementation is covered by targeted unit tests. The fix appears to be present on `main` after the latest release, so this is closeable as implemented-on-main with no shipped release tag yet.
