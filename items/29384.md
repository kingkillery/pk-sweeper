## Summary

Keep open. Current main has structured sandbox Docker config for image, network, user, caps dropped, env, resources, ulimits, security profiles, DNS, hosts, and binds, but it does not expose GPU, IPC, or arbitrary Docker create/run flags. The related #29933 closed Hostinger Docker socket/template issue is adjacent sandbox deployment work, not a duplicate of GPU/IPC sandbox parameters.
