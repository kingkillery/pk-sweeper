## Summary

Keep #29100 open. Current main still lacks a supported cost metadata overlay for auto-discovered Amazon Bedrock models. The related PR #31160 directly attempted the requested feature but was closed unmerged, and the current plugin-owned Bedrock discovery config still only supports enablement, region/filter, refresh interval, and default token limits.
