## Summary

Keep #27445 open. Current main has improved sub-agent completion routing and nested orchestration, but it does not implement the requested `announceTarget` config/default or `sessions_spawn` override for routing a top-level sub-agent completion to the parent session as an internal non-channel trigger. The remaining request is core sessions_spawn/announce behavior, not a ClawHub/plugin-only cleanup item, and related #33478 is complementary rather than a full canonical replacement.

## Evidence

It does not expose `announceTarget` or an equivalent documented routing option for native subagents.
