## Summary

Keep PR #60661 open. Current main still forwards `openclaw agent --timeout` as `params.timeout` into the gateway-owned `agent` request, so the observable cleanup requested by the PR is not implemented on main. The PR also still has an unresolved review concern from the provided discussion: its proposed accepted-ownership guard imports symbols that are not present in current `src/gateway/call.ts`, so the PR is useful but not ready to close as implemented or superseded.
