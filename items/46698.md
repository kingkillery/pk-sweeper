## Summary

Keep open. Current `main` does not implement this PR’s behavior: the GitHub Copilot device-flow login still polls immediately, still handles `slow_down` with a fixed `+2000ms` backoff, and the dedicated CLI command still has no `--wait` flag or plumbing for it.
