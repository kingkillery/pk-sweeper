## Summary

Keep open. Current `main` still uses the misleading `Downloading`/`Downloaded` Ollama onboarding copy, and the local-model merge/presence checks are exact-string only, so `gemma4` and `gemma4:latest` can still appear as separate entries and can still trigger an unnecessary pull when only `gemma4:latest` is installed.
