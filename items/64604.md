## Summary

Keep open. Current `main` and latest shipped release still collapse unclassified compaction failures to `unknown` in the `[compaction-diag]` log, so this is not implemented on `main` and not safely auto-closeable. The PR addresses a real observability gap, but it also carries unrelated README churn and still needs maintainer review rather than conservative cleanup closure.
