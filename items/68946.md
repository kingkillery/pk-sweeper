## Summary

Keep open. Current `main` includes an opt-in tool-loop detection system, but it is still disabled by default, and the tests explicitly confirm repeated tool calls are not blocked when that config is off. That means the issue's reported default behavior remains plausible on current `main`, so there is not enough evidence for a conservative close.
