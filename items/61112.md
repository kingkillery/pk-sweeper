## Summary

Keep open. Current `main` still hardcodes the markdown sidebar labels in English, so the request is not implemented; the submitted PR is also outdated against the Apr 24 sidebar rewrite and current generated-locale workflow.

  - file: [ui/src/ui/views/markdown-sidebar.ts:1](https://github.com/openclaw/openclaw/blob/78a431ed224b6f5b9d58672346551a5b411ad200/ui/src/ui/views/markdown-sidebar.ts#L1)
