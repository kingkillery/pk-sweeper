## Summary

Keep open. Current `main` still does not support the reported `bootstrapMaxCharsPerFile` key, so the core validation problem remains unresolved. Current docs do point users to `agents.defaults.bootstrapMaxChars` / `agents.defaults.bootstrapTotalMaxChars`, which narrows the issue to backward-compatibility or explicit migration/deprecation handling rather than missing documentation alone.
