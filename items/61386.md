## Summary

Current main already implements the heartbeat setTimeout overflow guard requested by PR #61386. The shipped implementation uses a shared safe timer-delay helper, adds a heartbeat scheduler regression test, and the v2026.4.24 changelog records the canonical fix via #71478.
