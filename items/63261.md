## Summary

Keep open. Current `main` still routes Discord owner/DM allowlist resolution through `account.config.dm?.allowFrom` only, while the config contract and normalization treat top-level `channels.discord.allowFrom` as the canonical alias. That mismatch matches the reported bug, and the same code is still present in the latest release tag `v2026.4.23`.
