## Summary

Current `main` and the latest shipped release already contain the webhooks/synchronous-registration fix described by the issue. The bundled `webhooks` plugin now registers synchronously, the loader rejects promise-returning `register()` hooks instead of silently dropping them, regression coverage exists, and `v2026.4.23` explicitly shipped `#67941` for this behavior.
