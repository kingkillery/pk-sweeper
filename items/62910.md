## Summary

Keep open. Current `main` and the latest release still contain the exact Windows `spawn(cmd, args, { shell: true })` / `spawnSync(cmd, args, { shell: true })` pattern described in the report, Node 24 is a supported recommended runtime, and PR #62910 is still an unmerged fix rather than obsolete work.

That is the DEP0190-triggering pattern the PR is addressing.
