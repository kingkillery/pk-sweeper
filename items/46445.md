## Summary

Keep open. Current main has substantial inbound debouncing infrastructure and per-channel config, but the repo docs and channel policy still describe the generic behavior as text-only. The issue explicitly asks for batching rapid-fire context messages including screenshots/voice/media, so the request is only partially implemented today.
