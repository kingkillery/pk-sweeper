## Summary

Keep open. Current `main` still renders assistant/web chat content through the Control UI markdown pipeline with plain `markdown-it` plus task-list handling only; there is no KaTeX/MathJax-style math renderer or math-specific hook in the WebUI. The report is recent, concrete, and fits the project’s stated web-frontend ergonomics work, so there is not strong evidence for any conservative close reason.
