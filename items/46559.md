## Summary

Keep open. On current `main` (`56fe2aab9c87596515f2280c8060dc740ea64a88`), the gateway still sends assistant chat deltas, flushed deltas, and final/error chat payloads to `nodeSendToSession(...)` unconditionally, so this PR’s stated verbose-off leak fix is not implemented on main.
