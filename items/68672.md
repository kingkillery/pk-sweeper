## Summary

Keep open. Current `main` still publishes `openclaw` through `openclaw.mjs`, whose Unix entrypoint is `#!/usr/bin/env node`, and this repo does not contain the `openclaw-env` wrapper/workaround named in the report. The Volta/version-manager handling present on `main` is limited to managed daemon/service install paths and docs warnings, not the raw CLI launcher described in this issue.
