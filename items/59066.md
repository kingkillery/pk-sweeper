## Summary

Keep open. The issue is recent, non-maintainer-authored, has no protected labels, and current main still contains the unsafe ACP projector helper/call path: `hashText(text: string)` returns `text.trim()` and the `usage_update` fallback calls `hashText(event.text)` without a local undefined guard. The timeline-linked PR #65567 is open, so the attempted fix has not landed on main.
