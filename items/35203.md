## Summary

Keep #35203 open. Current main has substantial subagent infrastructure, including isolated/forked child sessions, announce flow, nesting controls, runtime stats, and plugin runtime helpers, but the four RFC layers are not implemented as a whole and are not superseded by a single canonical tracker. The issue discussion itself maps related Octo/teleconference work as partial: Layer 1 partially staged, Layer 2 missing search/query/pub-sub, Layer 3 not shipped, and Layer 4 not enforced. Related open items cover narrower slices rather than the full RFC.
