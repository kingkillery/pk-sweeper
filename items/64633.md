## Summary

Keep open. Current `main` still treats `SOUL.md`/`USER.md` as injected context, but I did not find a provider-independent fix or regression coverage that proves the reported multi-turn style drift is resolved for the reporter's models (`Grok 4.1` / `Claude Sonnet`). The code/docs show intended behavior, not a verified fix, so this is not strong enough to close as implemented or cannot-reproduce.
