## Summary

Keep open. Current `main` does not implement time-bounded exec approvals: the shared approval decision model still only supports `allow-once`, `allow-always`, and `deny`; the `/approve` parser and approval action builders only handle those decisions; and the CLI/docs only expose persisted host policy plus allowlist entries, not expiring session approvals. This also still fits OpenClaw core scope because exec approvals are part of the project’s stated security/safe-defaults focus.
