## Summary

Keep open. The current main code still appears to match the reported gap: channel agent-tool factories are typed and invoked with only config, while the core OpenClaw tools receive requesterSenderId and senderIsOwner. The linked PR #58808 is also still open in the provided timeline, so this is not an already-landed cleanup close.
