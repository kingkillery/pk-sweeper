## Summary

Keep open. The PR still addresses a live gap on current `main`: `ExecApprovalsSocket.swift` still creates and accepts Unix sockets and writes through `FileHandle` without any `SO_NOSIGPIPE`/equivalent hardening, and the regression test proposed here is not present. The PR discussion is coherent and consistent with the current code shape, so there is not enough evidence to close it conservatively.
