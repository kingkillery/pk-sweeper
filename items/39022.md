## Summary

Keep open. Current `main` still treats `/model` as a persistent per-session override with no turn-budget argument, no session field for remaining turns, no `models.limits` config surface, and docs that only describe persistent `/model` selection.

## Evidence

  - file: [src/auto-reply/commands-registry.shared.ts:835](https://github.com/openclaw/openclaw/blob/d3dd61f2c5bd6cea175544f3a76a68ec11fa17c2/src/auto-reply/commands-registry.shared.ts#L835)
