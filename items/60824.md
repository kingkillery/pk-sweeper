## Summary

Keep open. Current main still has no ACP stream legacy migration/rule, while `acp.stream` is strict and only accepts the renamed current keys. The PR discussion also shows the one review concern was addressed on the PR head, so the remaining issue is that the useful fix has not landed and now needs maintainer review/rebase against the current migration files.
