## Summary

Do not close. Current `main` already excludes Codex from the OpenAI websocket path, but the rest of PR #56340 is not implemented on `main`: `before_tool_call` still has only terminal `block` semantics, approvals still lose to any block, docs/tests still describe the old behavior, and `loadProfileEnv()` still uses only `/bin/bash` rather than the proposed multi-shell fallback.
