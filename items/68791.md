## Summary

Keep open. Current `main` does not show a shipped fix for this report: native Windows docs still promise the exact local smoke path the reporter says now hangs, the embedded-runner still surfaces `payloads=0` incomplete turns as the same user-facing fallback, and current regression coverage explicitly leaves `stopReason=stop` plus zero-output turns out of scope rather than treating them as fixed.
