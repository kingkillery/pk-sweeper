## Summary

Keep open. Current main has several related surfaces, but they are partial: chat exposes session/model selectors, messages can show historical token/cost/model metadata, overview can show auth usage, and /status-style tooling can report model state. The requested always-visible interface/status area for the active agent plus model, token/quota, and subscription/auth context still appears to be missing.
