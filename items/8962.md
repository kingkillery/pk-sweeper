## Summary

Keep open. Current `main` has built-in Control UI/WebChat theme selection, but it does not expose the requested gateway-level custom CSS hook or any `gateway.webchat.theme`/`gateway.webchat.customCss` config surface. The request is only partially covered by existing appearance settings, so there is not strong evidence to close as implemented.
