## Summary

Keep PR #59852 open. Current main still has the LSP request timer leak the PR describes: request timeouts are created without retaining the handle, successful responses delete the pending request without clearing the timer, and session disposal rejects pending requests without clearing timers. The PR remains a small, relevant fix rather than an item that can be closed as implemented, duplicate, stale, or out of scope.

## Evidence
