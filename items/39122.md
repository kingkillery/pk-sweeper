## Summary

Keep open. Current `main` already added stronger Windows scheduled-task restart cleanup that force-releases the gateway port before `/Run`, but the PR’s separate `runDaemonRestart` change to skip the Windows stale-PID retry is still not present. That means the repo has partial overlap with the PR, not a clear implemented-on-main match.
