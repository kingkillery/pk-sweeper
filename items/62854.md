## Summary

Keep open. Current `main` contains partial mitigations for the reported `update_plan` problems, but it still auto-enables the tool for strict-agentic GPT-5 OpenAI/Codex runs, and the issue timeline already links a newer open bug reporting similar `update_plan` burst behavior on recent builds. That is not strong enough evidence to close conservatively.
