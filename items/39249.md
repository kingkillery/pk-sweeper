## Summary

Current `main` already implements the requested housekeeping paths: `doctor` detects orphan session transcript files and can archive them as `.deleted.<timestamp>`, `openclaw sessions cleanup --fix-missing` prunes `sessions.json` entries whose transcript files are gone, and disk-budget enforcement removes orphan/archived transcript artifacts first when cleanup runs.
