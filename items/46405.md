## Summary

Keep open. Current `main` already has the gateway `config.patch` hot-reload behavior, but it does not implement the MiniMax XML tool-call promotion this PR proposes; current code and tests still strip MiniMax `<invoke ...></invoke></minimax:tool_call>` content instead of turning it into structured tool-use blocks.
