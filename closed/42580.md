## Summary

Current `main` has moved to a different, intentionally canonical edit-tool contract. The PR’s target code path (`CLAUDE_PARAM_GROUPS` / `normalizeToolParams` / schema alias patching) is gone, `edit` now requires `path` plus `edits[]`, and current tests explicitly reject legacy alias inputs like `file_path` and `old_text`. This makes the PR obsolete against current `main` rather than a merge candidate.
